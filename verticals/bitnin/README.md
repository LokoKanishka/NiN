# BitNin

BitNin es la vertical formal de NiN para inteligencia historica de mercado, memoria RAG, analisis estructurado y simulacion futura.

La fuente de verdad de esta vertical esta en `verticals/bitnin/SSOT.md`.

BitNin v1 no toca dinero ni ejecuta trading real. La vertical queda en fase fundacional y shadow-first.
