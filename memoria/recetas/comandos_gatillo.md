# 🎯 Comandos Gatillo Oficiales

Estos son los comandos de voz o texto que disparan acciones automáticas en Antigravity.

| Comando | Acción |
|---|---|
| **"cunn protocolo de inicio"** | Ejecuta el script `startup_check.py` completo (Salud, Red, Doctor, Memoria). |
| **"inicio"** | Ejecuta el script `startup_check.py` completo. |
| **"inicia"** | Ejecuta el script `startup_check.py` completo. |
| **"doctor"** | Ejecuta `Tool: Doctor System` para reparar contenedores Docker. |
| **"salud"** | Ejecuta `Tool: System Health` para diagnosticar hardware. |

> [!TIP]
> Estos comandos están diseñados para ser cortos y fáciles de decir por voz o escribir en el chat.
